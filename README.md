<h1><b>ThinkNotes : A Task Manager</b></h1>
<p>ThinkNotes is an easy-to-use task manager made for looking up to your overall workloads or tasks.
It can create, update and delete tasks according to your inputs and needs and have a very clean UI/UX design.</p>
<hr>
<h2>*****Frontend*****</h2>
<p>The whole frontend is made using React framework with designs and themes taken from daisyUI.Tailwind CSS was used somewhere for styling.
It has very simple features of:
<ul>
<li>Displaying Tasks
<li>Creating Tasks
<li>Updating Status
<li>Deleting Tasks
</ul>
